jQuery 개요

        HTML 문서 조작, 이벤트 처리, 애니메이션 그리고 Ajax를
        훨씬 쉽게 할 수 있도록 도와주는 빠르고, 작고 
        기능이 풍부한 javaScript 라이브러리

        모토 : write Less, Do More(더 적게 쓰고, 더 많이 하라)
        HTML / DOM 조작 : 웹페이지의 특정 요소를 쉽게 찾고, 내용을 바꾸거나
                          스타일을 변경하고, 새로운 요소를추가하거나 삭제할 수 있음
                          예) 특정 버튼을 누르면 글자색을 변경 할 수 있음

        이벤트 처리      : 사용자의 클릭, 마우스 오버, 키보드 입력 등
                          다양한 이벤트를 간단하게 처리할 수 있는 코드 제공

        애니메이션 효과 : 웹 페이지 요소들을 부드럽게 사라지게 하거나(fade out)
                                                   나타나게 하고  (fade in)
                        슬라이딩 시키는 등의 다양한 애니메이션 효과를 구현할 수 있다.

        Ajax(Asynchronous JavaScript and XML ) :
                        페이지 전체를 새로고침 하지 않고도 서버와 데이터를 
                        주고 받아 동적으로 페이지의 일부를 업데이트 할 수 있게 해줌
                        예) 댓글 작성했을때 페이지 이동 없이 바로 댓글이 추가
                            로그인 했을 경우 페이지 새로고침 없이 로그인 공간만
                            로그인/ 로그아웃 상태 변경 가능

기본 문법

        
            $(선택자).동적함수();
        
        $ : jQuery를 의미하는 기호
            이제부터 jQuery 문법을 시작하겠습니다.라는 신호와 같음
        (선택자) : 동작을 적용하고 싶은 HTML 요소 선택
                    class = .    id = #     tag = tag 명칭
                    css 선택자와 거의 동일한 방식 사용
                    $("p") : 모든 <p> 태그를 선택
                    $("#myId") : id가 myId인 요소를 선택
                    $(".myClass") : class 가 myClass인 모든 요소 선택
        동적함수(Action) : 선택된 요소에 실행할 동작 지정
                    .hide() : 요소를 숨김
                    .show() : 요소를 보여줌
                    .text("새로운 글자") : 요소의 텍스트를 변경
                    .css("color", "red") : 요소의 css 스타일을 변경
                    .val() : input select 같은 폼 요소의 값을 가져오거나 변경
                    .attr("속성명칭","값") : 요소의 속성 변경

$(document).ready()

        jQuery 코드는 동작하려는 html 요소가 화면에 모두 그려진 다음에 실행되어야 안전
        만약 HTML이 만들어지기도 전에 jQuery가 특정 요소를 찾으려하면 오류가 발생
        모든 jQuery 코드는 $(document).ready() 함수 기능 안에 작성하는 것이 기본 규칙

        고전방법
        
            $(document).ready(function(){
                // 이 안에 모든 jQuery 코드를 작성
                // 이 코드는 HTML 문서가 완전히 준비된 후 실행
            })
        

        최신방법
        
            $(function(){
                // 이 안에 모든 jQuery 코드를 작성
                // 이 코드는 HTML 문서가 완전히 준비된 후 실행
            })
