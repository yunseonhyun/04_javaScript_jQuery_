애니메이션 / 기본 시각 효과 메서드

기본 시각 효과 메서드

        fadeIn(ms시간 초) / fadeOut(ms시간 초) : 투명도 변화로 나타남/사라짐
        * ms시간 초 = 500 = 0.5초, 1000 = 1초
            ex) : .fadeIn(500) : 0.5초 동안 부드럽게 나타남
        slideDown() / slideUp() : 슬라이드 효과로 펼침/접힘
        show()      / hide()    : 기본 표시 / 숨김
        toggle()                : 상태에 따라 보이기 / 숨기기 토글
        animate()               : css 속성을 단계적으로 애니메이션 처리
            * width, height, opacity, margin, top, left 등 조작 가능
            ex) : .animate({ css 속성 : 속성값 들 }, 시간, 시간 후에 작업할 기능, 스타일 설정)
