.attr() 메서드

        HTML 요소의 속성(attribuye)을 조작하는 메서드(기능 = 함수 = 행동)
        주로 <a> 태그나 <img>태그에 많이 사용하나
        모든 태그에 사용 가능

        1. 속성값 가져오기
        $("선택자").attr("속성명");

        2. 속성값 설정하기
        $("선택자").attr("속성명","값");

        3.여러 속성 동시 설정
        $("선택자").attr('속성명1', '값1').attr('속성명2', '값2');

기본 자바스크립트 attr 속성

        
        1. 속성값 가져오기 .getAttribute();
        document.querySelector("선택자").getAttribute("속성명");

        2. 속성값 설정하기 .setAttribute()
        document.querySelector("선택자").setAttribute("속성명","값");

        3.여러 속성 동시 설정
        document.querySelector("선택자").setAttribute("속성명1","값1").setAttribute("속성명2","값2");

        4.속성제거
        document.querySelector("선택자").removeAttribute('속성명','값');
        document.querySelector("선택자").removeAttribute('속성명');
