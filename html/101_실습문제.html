👤 Random User API 실습
사용자 정보를 불러오는 중...

🎯 버튼을 클릭해서 랜덤 사용자를 생성해보세요!

Random User API에서 실제 데이터를 가져와 표시합니다.

🎯 구현해야 할 기능들:

1. 사용자 가져오기: $.get()으로 API 호출하여 데이터 가져오기

2. 데이터 표시: 가져온 데이터로 사용자 카드 생성

3. 사용자 지우기: 현재 표시된 사용자 카드 제거

💡 API 정보:
• API URL: https://randomuser.me/api/
• 응답 데이터: data.results[0]로 첫 번째 사용자 접근
• 이미지: user.picture.large
• 이름: user.name.first + " " + user.name.last
🔧 필요한 jQuery 메서드:
• $.get(url, function(data) { ... }) - API 호출
• $("#userCard").html(content) - 내용 업데이트
• $("#userCard").fadeIn(300) - 애니메이션 표시
• $("#userCard").fadeOut(300) - 애니메이션 숨김
