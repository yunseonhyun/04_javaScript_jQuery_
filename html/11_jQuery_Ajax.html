Ajax와 jQuery

Ajax 개념

        웹 페이지를 새로고침 하지 않고도 서버와 데이터를 주고받을 수 있는 기술
        비동기적으로 작동하여 사용자가 페이지르 사용하는 동안
        백그라운드에서 서버 통신이 이루어짐

        - 로그인 / 로그아웃
        - 실시간 검색 기능
        - 현재 페이지에서 데이터를 전송할 때 사용

jQuery에서 Ajax 사용하기

        jQuery는 Ajax를 매우 간단하게 사용할 수 있도록 도와주는 라이브러리
        복잡한 js코드를 간단한 메서드로 대체할 수 있다.

주요 jQuery Ajax 메서드

$.ajax() 기본 메서드

$.ajax({ url:'/데이터가 저장된 주소' type:'GET' dataType:'json' success : function(data){ console.log("대아터 가져오기 성공 : ", data); }, error : function(xhr, status, error) { console.log("문제가 발생하였습니다. : ", error); } }) 
        xhr : XmlHttpRequest
              xhr.status     //    404,       500,        403     등 문제 발생
              xhr.statusText // not found  serverError  권한없음
        status : 오류의 종류를 나타내는 문자열
                 timeout - 요청시간초과 error - HTTP 오류(4XX, 5XX 상대코드) 
                 abort - 요청 중단   parsererror - 응답데이터 파싱 실패(JSON 변환 오류 등) 
        error :  구체적인 설명을 담을 문자열

$.get()

        GET 요청을 간단하게 보낼 때 사용
$.get('데아터가져올주소', funxtion(data) { // 데이터 가져온 후 실행할 기능 작성 } )

$.post()

        POST 요청으로 데이터를 서버에 전송할 때 사용
$.post('데아터를 백엔드로 보내서 저장할 주소', { // 키 값 형태로 데이터를 백엔드로 전달할 때 전달할 값을 담을 키와 키 내부에 담을 값을 작성 "키" : "값", "키" : "값", }, funxtion(response) { // 데이터 저장한 후 실행할 기능 작성 } )
