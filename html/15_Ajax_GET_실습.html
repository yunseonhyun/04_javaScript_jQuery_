jQuery AJAX $.get() 실습문제 10개

각 문제를 클릭해서 AJAX 요청을 실습해보세요!

1: 기본 텍스트 데이터 가져오기

JSONPlaceholder API에서 첫 번째 게시물 제목을 가져와 보세요.

2: 사용자 정보 표시하기

사용자 ID를 입력하면 해당 사용자의 이름과 이메일을 표시하세요.

3: 랜덤 명언 가져오기

Quotable API에서 랜덤 명언을 가져와 표시하세요.

4: 로딩 상태 표시하기

데이터를 가져올 때 로딩 메시지를 표시하고, 완료되면 숨기세요.

5: 에러 처리하기

존재하지 않는 URL로 요청을 보내고 에러를 처리하세요.

문제 6: 게시물 목록 가져오기

여러 게시물을 가져와서 목록으로 표시하세요.

7: 공개 API 활용하기

httpbin.org API를 사용해서 현재 IP 정보를 가져오세요.

문제 8: 검색 기능 구현하기

사용자 이름으로 검색하여 해당 사용자를 찾으세요.

문제 9: 선택된 항목에 따른 데이터 가져오기

앨범 ID를 선택하면 해당 앨범의 사진들을 가져오세요.

10: 종합 실습 - 대시보드

여러 API를 동시에 호출해서 간단한 대시보드를 만드세요.
